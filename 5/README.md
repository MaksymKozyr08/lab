# Лабораторна робота: Алгоритми на графах

## Мета роботи
Реалізація базових та просунутих алгоритмів на графах з використанням принципів об'єктно-орієнтованого програмування на C++. Дослідження та порівняння ефективності різних способів подання графу (матриця суміжності та списки суміжності).

## Виконані завдання
У лабораторній роботі реалізовано наступний функціонал:
1. **Базова архітектура (ООП)**: Створено загальний інтерфейс `IGraph`, від якого успадковані класи `GraphMatrix` (матриця суміжності) та `GraphList` (списки суміжності). Алгоритми працюють з абстракцією `IGraph`.
2. **Блок 1**: Перевірка графу на зв'язність та пошук усіх компонент зв'язності.
3. **Блок 2**: Обхід графу в глибину (DFS) та обхід в ширину (BFS). Реалізовано додаткову можливість сортування сусідів за вагою ребра перед відвідуванням.
4. **Блок 3**: Алгоритми пошуку найкоротших шляхів — алгоритм Дейкстри (з пріоритетною чергою) та алгоритм Флойда-Воршелла.
5. **Блок 4**: Топологічне сортування спрямованого ациклічного графу (DAG) двома методами: алгоритмом Кана та на основі обходу DFS.
6. **Блок 5**: Знаходження та побудова кістякового дерева на основі алгоритму BFS.
7. **Блок 6**: Побудова мінімального кістякового дерева (MST) для зважених неорієнтованих графів за допомогою алгоритмів Пріма та Крускала (з ручною реалізацією структури Disjoint Set Union - DSU).
8. **Візуалізація**: Додано метод `visualize()`, який зберігає дані графу у текстовий файл, генерує скрипт на Python і автоматично запускає його для малювання графу за допомогою бібліотек `networkx` та `matplotlib`.

## Структура проекту
- `main.cpp` — єдиний вихідний файл з реалізацією всієї логіки, класів і алгоритмів.
- `graph_data.txt` — проміжний файл, що створюється автоматично для обміну даними між C++ та Python.
- `visualize.py` — автоматично згенерований скрипт для малювання графу.

## Режими роботи (меню)
Програма містить 3 повноцінні режими, які можна обрати при запуску:
1. **Demo (Демонстраційний)**: Жорстко задається граф у вигляді матриці, який конвертується у списки суміжності. Відбувається покроковий виклик реалізованих алгоритмів із виведенням результатів на екран, після чого запускається графічна візуалізація.
2. **Interactive (Інтерактивний)**: Користувач власноруч вводить кількість вершин, ребер та ваги. Після створення графу з'являється меню для виконання будь-якого алгоритму з лабораторної на вибір.
3. **Benchmark (Тестування швидкодії)**: Програма генерує великий розріджений граф (наприклад, 1000 вершин, 5000 ребер) і заміряє час виконання алгоритму Дейкстри окремо для матриці суміжності та списків суміжності за допомогою бібліотеки `<chrono>`.

## Інструкція із запуску

### 1. Компіляція C++
Для компіляції використовуйте компілятор `g++` у терміналі (у папці з проектом):
```bash
g++ main.cpp -o lab_graphs.exe
```

### 2. Запуск програми
Після успішної компіляції запустіть файл:
```bash
./lab_graphs.exe
```

### 3. Налаштування візуалізації (опціонально)
Щоб програма змогла намалювати граф, на вашому комп'ютері має бути встановлений Python, а також бібліотеки для візуалізації. Встановіть їх через pip, якщо вони відсутні:
```bash
pip install networkx matplotlib
```
