# Лабораторна робота №6: Впорядковані списки та дерева

## Опис завдання
**Варіант**: 3D координати точок, сортування за відстанню від початку координат.

Проєкт демонструє роботу різних структур даних (списків та дерев) із сортуванням точок. Головною фішкою є можливість покрокової візуалізації процесу додавання елементів через Python-скрипт з графічним інтерфейсом.

## Виконані завдання

1. **Реалізація структур даних (ООП підхід)**:
   - Усі структури успадковуються від шаблонного інтерфейсу `IOrderedList` (файл `IOrderedList.h`).
   - **ArrayList**: реалізація на базі динамічного масиву зі зсувом елементів при вставці.
   - **LinkedList**: власна реалізація однозв'язного впорядкованого списку.
   - **BST**: класичне бінарне дерево пошуку без балансування.
   - **AVLTree**: збалансоване бінарне дерево з підтримкою LL, RR, LR, RL поворотів.
   - **StdSetWrapper**: обгортка над бібліотечним `std::set` (який під капотом працює як Red-Black Tree) для додаткових балів за використання стандартної бібліотеки.
   *(Усі реалізації знаходяться у файлі `DataStructures.h`)*.

2. **Логіка сортування точок**:
   - Структура `Point3D` (файл `Point3D.h`) містить перевантажені оператори порівняння (`<`, `>`, `==`). 
   - Порівняння відбувається за **квадратом відстані** до початку координат (`x*x + y*y + z*z`), що дозволяє уникнути проблем із втратою точності при обчисленні кореня.

3. **Візуалізація та логування**:
   - Написано окремий скрипт `visualizer.py` для відмальовки графіки.
   - C++ програма автоматично зберігає всі кроки у форматі JSON у файл `steps.json`.
   - Python зчитує ці дані та за допомогою `matplotlib` і `networkx` малює інтерактивне вікно з кнопками **Next/Prev** і перемикачем структур даних.

## Режими роботи (`main.cpp`)

Програма має текстове меню з 3 режимами:
1. **Демо-режим**: Програма самостійно генерує 10 рандомних точок, проганяє їх через усі 5 структур, логує процес та автоматично запускає візуалізатор.
2. **Інтерактивний режим**: Користувач власноруч вводить координати точок (X Y Z). Після завершення вводу відкривається візуалізація побудованих дерев.
3. **Бенчмарк (Тест на швидкість)**: Генерується 10 000 випадкових точок і заміряється час вставки в усі структури за допомогою `std::chrono` (швидкість відпрацювання O(N) в ArrayList проти O(log N) в AVL).

## Інструкція із запуску

### 1. Підготовка середовища
Для роботи візуалізатора потрібен встановлений Python. Встановіть необхідні бібліотеки командою:
```cmd
pip install matplotlib networkx
```

### 2. Компіляція
Перейдіть у папку з проєктом та скомпілюйте `main.cpp` (рекомендовано стандарт C++17):
```cmd
g++ main.cpp -o lab6.exe -std=c++17
```

### 3. Запуск
Запустіть зібраний `.exe` файл:
```cmd
.\lab6.exe
```
Після вибору Демо- або Інтерактивного режиму програма автоматично запустить `visualizer.py`, і ви побачите графічне вікно з ієрархією вузлів.
